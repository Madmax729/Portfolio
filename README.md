## 🏗️ About This Portfolio

This portfolio is built with modern web technologies and designed for a **premium, immersive experience**:

| Feature | Technology |
|---------|-----------|
| ⚡ Framework | **Next.js 13** (App Router) |
| 🎨 Styling | **Tailwind CSS** |
| 🎬 Animations | **Framer Motion** + **GSAP** |
| ✍️ Typography | **Syne** + **Mona Sans** |
| 🖱️ Cursor Effects | **Blobity** |
| 📊 Analytics | **Vercel Analytics** |
| 🚀 Deployment | **Vercel** |

### Key Sections
- 🏠 **Hero** — Animated intro with profile & social links
- 💼 **Work** — Featured projects with alternating card layout
- 📝 **Publications** — Research papers with IEEE links
- 🙋 **About** — Bio, expertise & a song carousel
- 🛠️ **Skills** — Categorized skill grid with animated cards
- 📰 **Blog** — Latest articles & writeups
- 📬 **Contact** — Let's talk section with email & booking

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Madmax729/Portfolio.git

# Navigate to the project
cd Portfolio

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
